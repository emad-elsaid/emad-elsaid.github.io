# Books Library

When the user says anything that matches a reading status update — such as:

- "I'm reading / started / currently reading X"
- "I finished / completed / done with X"
- "I want to read X"
- "I didn't like / gave up on / couldn't finish X"

…treat it exactly as if they typed `/book <their message>` and invoke the `book` skill immediately, without asking for confirmation.
